Project Overview

This project involves an in-depth exploratory data analysis of the famous Titanic dataset, examining factors that influenced passenger survival rates.

About the Analysis

This exploratory data analysis delves into the Titanic dataset to understand the factors that contributed to passenger survival. The analysis includes data cleaning, feature engineering, visualization, and statistical analysis.

Key findings reveal correlations between survival rates and factors such as passenger class, gender, age, and fare paid. The analysis provides valuable insights into one of history's most tragic maritime disasters.

The project showcases strong data manipulation and visualization skills using Python's data science ecosystem, including Pandas, NumPy, Matplotlib, and Seaborn.

Data Visualization

Created insightful visualizations to represent patterns and correlations in the data.

Statistical Analysis

Performed statistical tests to validate hypotheses about factors affecting survival.

Data Cleaning

Handled missing values, outliers, and inconsistent data to ensure analysis accuracy.

Insightful Conclusions

Drew meaningful conclusions from the data that provide value beyond the numbers.
